# Sentetik Veri Örnekleri

Bu dizindeki dosyalar (örneğin `sample_cicids2017.csv`) tamamen sentetik olarak üretilmiştir. Gerçek kullanıcı veya ağ trafiği içermezler.

Dosyalar, CIC-IDS2017 veri setine uygun olarak 78 özellik ve opsiyonel 'Label' sütunu yapısını göstermek, geliştirme, test ve demonstrasyon (demo) amacıyla kullanılmak üzere oluşturulmuştur.

Önemli Uyarı:
- Bu dosyalar model performansı ölçümü, makine öğrenmesi eğitimi veya gerçek saldırı analizi için KULLANILMAMALIDIR.
- Gerçek CIC-IDS2017 ham (raw) veya işlenmiş veri dosyaları, büyük boyutları ve lisans koşulları nedeniyle GitHub repository'sine yüklenmemelidir.
